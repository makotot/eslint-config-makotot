## Install

```sh
$ npm i --save-dev eslint-config-makotot
```

## Usage

In `.eslintrc`:
```
{
  "extends": ["eslint-config-makotot"]
}
```
or in `package.json`:
```json
"eslintConfig": {
  "extends": ["eslint-config-makotot"]
}
```
